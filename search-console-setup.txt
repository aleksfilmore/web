# Google Search Console verification
# Add this meta tag to your homepage head section:
# <meta name="google-site-verification" content="[YOUR_VERIFICATION_CODE]" />

# Bing Webmaster Tools verification  
# Add this meta tag to your homepage head section:
# <meta name="msvalidate.01" content="[YOUR_VERIFICATION_CODE]" />

# Instructions for setting up:
# 1. Go to Google Search Console (search.google.com/search-console)
# 2. Add your property (https://aleksfilmore.com)
# 3. Get your verification meta tag
# 4. Add the meta tag to index.html in the <head> section
# 5. Verify ownership
# 6. Submit your sitemap.xml URL: https://aleksfilmore.com/sitemap.xml

# Same process for Bing Webmaster Tools:
# 1. Go to webmaster.bing.com
# 2. Add your site
# 3. Get verification meta tag
# 4. Add to index.html
# 5. Verify and submit sitemap
